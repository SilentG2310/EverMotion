# 🚀 EverMotion | AI-Driven Rehabilitation Ecosystem

> **Rehab built for Movement.**
> A hardware-free, AI-powered hand tracking system designed to make physical therapy seamless, engaging, and accessible.

---

## 📖 Project Overview

**EverMotion** is an innovative rehabilitation platform that breaks down the hardware barriers of traditional therapy. By leveraging **MediaPipe Hands** technology, patients can perform clinical-grade exercises using only a standard webcam. The system features a custom **Adaptive AI** engine that calibrates in real-time to match each user's unique range of motion and physical capabilities.

## ✨ Key Features

* **Hardware Free**: Eliminates the need for expensive sensors or wearable gloves by using advanced computer vision.
* **Adaptive AI**: Automatically adjusts gameplay difficulty and interaction zones based on the user's specific motor skills.
* **Clinical Design**: Developed with a focus on neuroplasticity, coordination, and precision control exercises.
* **Modern Interface**: Built with a **Solana Ultra Dark** aesthetic, featuring "flexible bubble" UI components and smooth glassmorphism effects.

## 🛠 Tech Stack

* **Frontend**: HTML5, CSS3 (featuring Ambient Glow effects and responsive Bento Grid layouts).
* **Interaction**: Vanilla JavaScript (Ambient parallax, side-drawer data injection, and fluid accordion transitions).
* **AI Engine**: MediaPipe Hands (Real-time 21-point hand landmark tracking).
* **Design System**: Commercial Tech Style V4.0 (Blur-heavy backgrounds and rounded "bubble" geometry).

## 📁 Project Telemetry (Archive)

## 🚀 Quick Start

1.  **Clone the Repository**: Download the project files to your local machine.
2.  **Launch Environment**: Open `index.html` via a local server (e.g., VS Code **Live Server**) to ensure all AI scripts load correctly.
3.  **Calibrate & Play**: Click **"Explore Games"** on the home page to begin the AI calibration process.
